## Visão geral do projeto

**BoraJogar** é um app React Native + Expo para organização de eventos esportivos (principalmente futsal). Permite gerenciar grupos/times, convocações, times por partida, enquetes e estatísticas.

- **Frontend:** React Native 0.81.5 + Expo ~54.0.33 + Expo Router ~6.0.23
- **Backend:** Supabase (PostgreSQL 17 + Auth + Storage + Edge Functions)
- **Build/Deploy:** EAS (Expo Application Services)

## Problema identificado: Docker não está rodando

O ambiente local do Supabase (via CLI) exige o **Docker Desktop** ativo no Windows. Sem ele, comandos como `supabase start`, `supabase db reset` e `supabase functions serve` falham com:

```
open //./pipe/docker_engine: O sistema não pode encontrar o arquivo especificado
```

**Solução:** Abrir o Docker Desktop e aguardar o daemon inicializar antes de usar o CLI local.

## Autenticação

- Fluxo: email/senha com confirmação OTP
- Deep links: `borajogar://login`, `borajogar://reset-password`
- Perfil criado automaticamente via trigger no banco ao cadastrar usuário
- Papéis globais: `super_admin`
- Papéis por conta: `group_admin`, `group_moderator`, `player`
- RLS habilitado com funções helper:
  - `current_user_is_super_admin()`
  - `current_user_has_account_access()`
  - `current_user_has_account_role()`
  - `current_user_can_access_event()`
  - `current_user_can_manage_event()`
